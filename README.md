# PHP E-commerce Project (Route Academy)

This is a Laravel-based e-commerce platform built for Route Academy.

## Key Features I Implemented

1. Continue shopping button on the cart page.
2. Proceed to checkout functionality.
3. Google login integration.
4. Product detail pages.
5. Remove products from cart and wishlist.
6. Redesigned pages for a cleaner, more professional look.
